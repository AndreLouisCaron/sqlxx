Detailed Description

SQL Statement.

The statement, in SQL parlance, is the premium mechanism of communication between a client application and a database. Each one represents a single operation from the client side, including any reply the database emits.

sql::Statement::Statement ( Connection &  connection)

Create a Statement for a Connection.
