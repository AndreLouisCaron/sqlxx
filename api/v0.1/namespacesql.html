typedef ::SQLWCHAR sql::wcharacter

UTF-16 16-bit unit.

It is guaranteed that 16-bit units cannot always represent a character. A single unit cannot represent code points outside the basic multilingual plain and cannot handle decomposed characters. Moreoever, some code points may represent more than one "user-perceived character".
